cold-cli vision: build the infrastructure layer that powers AI sales agents' outbound. Not just a cold email CLI — an outbound execution engine for agents.

**Why:** AI agents are the future of outbound sales. Every agent needs to send messages, handle replies, manage deliverability, optimize sequences. Today they hack together Gmail API calls. cold-cli becomes the Twilio/Sendgrid for agent-powered outbound.

**How to apply:** Frame all architectural decisions around agent-first design. The CLI is one interface; the API is the primary product. Cloud reliability is non-negotiable for agents. Multi-provider support is table stakes. Think platform, not tool.

Target: YC-fundable, venture-scale ($100K+/mo → $1M+/mo).
